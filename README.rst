==========
Cipr Class
==========

Classes for Corona SDK

Installation
============

Installation is done with `cipr <http://github.com/six8/corona-cipr>`_

::

	cipr install:git://github.com/six8/cipr.class.git

Usage
=====

::

	local cipr = require 'cipr'
	local class = cipr.import('cipr.class')
